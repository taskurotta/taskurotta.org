Taskurotta uses Hazelcast Framework for creating shared memory and runtime environment between Taskurotta servers. This framework allows us to create transparent scalability. All nodes have auto-discovery feature which helps to register new node and distribute memory from one node to another.

Main features:

  • Dynamic process creation
  • Simple run and control of processes
  • Reusable Actors in processes(Workers and Coordinators)
  • Regression tests based on actors and archive data
  • Handy creation of actors
  • Handy scalability of actors and servers
  • Fault-tolerance and load-balancing
  • Process Scheduler
