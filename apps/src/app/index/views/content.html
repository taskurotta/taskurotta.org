Taskurotta uses Hazelcast Framework for creating shared memory and runtime environment between Taskurotta servers. This framework allow us create transparent scalability. All nodes has auto-discovery feature which helps register new node and distribute memory from other nodes on this node.

Main features:

  • Dynamic process creation
  • Simple run and control of processes
  • Reusable Actors in processes(Workers and Coordinators)
  • Regression tests based on actors and archive data
  • Handy creation of actors
  • Handy scalability of actors and servers
  • Fault-tolerance and load-balancing
  • Process Scheduler
